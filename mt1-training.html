第一部分：MT1 核心产品技术与功能详解

1.1 产品定位与核心场景

  • 目标客户：对清洁质量与标准有较高要求的中高端客户（如高端酒店宴会厅、大型仓库、购物中心、机场），而非仅追求低价的三星级标准客户。
  • 核心优势场景：面积大（4,000 – 10,000 m²）、重复性高、清洁时间窗口紧迫（如酒店宴会厅凌晨布展后）、人力成本高或人力不稳定的场景。
  • 核心价值主张：清洁效果与一致性（超越人工视觉判断）、效率提升（实际 ~1,500 m²/h vs 人工 ~500 m²/h）、管理成本降低（数字化报告可追溯）。

1.3 清洁工作原理

  • 双边刷将垃圾聚拢到中央 → 中央粗滚刷卷抛进入垃圾盒（hopper）。
  • 后部吸尘模块通过双风机形成负压，吸入细微粉尘、毛絮等不可见颗粒，经 HEPA 滤网过滤后排出。
  • 滤网分 HEPA 11（标配）与 HEPA 13（可选，更密但吸力衰减）。滤网为可水洗复用件，约 3 个月 / 10 个 cycle 更换一次。
  • 控尘风机：即便是扫地模式（不吸尘）也有小风机形成流场控制扬尘。

1.4 硬件与传感器系统（安全保障）

  • 定位：单线激光雷达（建图/定位）+ 视觉天花板定位（VSLAM）+ Marker，多地图可拼接。
  • 主动避障：RGB-D（3 路：正前下视 + 侧面）+ 激光雷达融合。默认 50 cm 内减速并尝试绕行，无法绕行时停车等待。
  • 全方位感知：前视用于避障、后视用于垃圾识别（AI Trash Recognition）。
  • 碰撞传感器：用于检测雷达盲区的悬空障碍（如桌沿、悬挂物），悬空障碍物处理高度约 52 cm。
  • 多线激光雷达（MT1 Pro 配置）：用于停车场等高速车辆场景，提高响应速度；可穿透部分黑色反射物。
  • 认证：产品已获得 UL 安全认证；充电桩（charging station）另有整套安规认证。
  • 通过性：可处理常见小坡度，无法应对楼梯；可加装通讯模块与电梯联动（改造电梯即可替代人按钮）。

1.5 清洁能力边界与注意事项

  • 尺寸限制：中央滚刷入口高约 10 cm，过高的直立瓶子会卡住并触发避障；躺倒的塑料瓶可清扫，宽度通道 40 cm。
  • 液体 / 湿垃圾：MT1 不支持，识别到液体会避障；可通过系统"召唤"CC1 / CG one 前来洗地（干湿协作）。
  • 缠绕物：打包带、长条物会缠绕边刷与滚刷，我们默认加入避障；仓库中的小块泡沫塑料会被直接吸入，但体积较大时可能搅入滚刷。
  • 极细粉末 / 木屑：基础扫地款（非 MT1）的 hopper 较小易满，需用 MT1 的 35 L 大 hopper；对面粉级别的粉末 MT1 表现最佳。
  • 耗材磨损：滚刷、边刷、胶条、HEPA 滤网为消耗品，磨损速度取决于使用频率与地面粗糙度。粗糙地面约 3 – 4 个月换一次边刷，光滑地面可延长到 6 个月。

第二部分：客户演示策略与销售话术

2.1 演示前准备：了解客户与场景

  • 询问关键信息：现场区域划分、现有清洁需求、人员数量 / 排班、当前痛点（效果、效率、成本、管理难度）。
  • 识别决策者：重点影响老板 / 管理者——关注 ROI、清洁标准、管理效率。无需过度担心基层员工的接受度（他们的 resistance 后续阶段再处理）。
  • 筛选目标场景：引导客户关注大面积、高重复、夜班清洁等 MT1 优势最明显的场景（酒店宴会厅、仓库主通道、商场大堂）。

2.2 演示核心流程：五大成功因素

本次培训总结的五大客户说服因素（按客户类型取舍讲解）：

因素话术要点
① 效果机器吸尘效果可视化超越人工：打开垃圾盒 / 尘袋展示收集到的灰尘。
② 效率人工 ≈ 500 m²/h（且疲劳衰减）；MT1 理论 1,800 m²/h，实际 ≈ 1,500 m²/h。
③ 一致性机器每天稳定输出同样质量；人工随情绪、疲劳波动。

⑤ 管理成本减少 supervisor 督导工时，签到表 / 手动巡检的管理成本降低。

2.3 四步演示法：从建立信任到展示价值

第一步：打消安全顾虑（Safety First）

主动展示传感器系统（激光雷达、RGB-D、碰撞传感器），现场验证机器人对突然出现的行人 / 障碍物的避让行为，强调 UL 认证。

第二步：凸显清洁效果（Effectiveness）

选择客户现场看似干净但实际易脏的区域（酒店门口地毯、走廊），使用自带粉末（面粉、爽身粉、咖啡渣）或模拟碎屑进行演示；清洁后打开垃圾盒 / 尘袋，直观展示收集物。

关键：放慢速度、撒少量、撒一溜而非一大片——速度慢时吸尘效果最好。

第三步：算清效率与经济账（Efficiency & ROI）

了解客户现有人工清洁该区域所需时间和人数，对比 MT1 理论 / 实际效率。美国人力成本参考：清洁工约 $2,000 – 3,000 / 月；supervisor 约 $6,000 / 月（一个 supervisor 最多管 3 – 6 人）。MT1 整机约 $19,000 + 服务包 + 耗材（打包价约 $20,000）。替换 1 – 2 名全职清洁工通常可在一年内回本。

第四步：展示智能与便捷（Intelligence & Management）

演示快速建图、清洁区域与路径设置、不同区域排程，并展示可导出的清洁报告（coverage、trash map、任务时长）；强调远程数字化管理优势。

2.4 应对客户关切（Handling Concerns）

  • "太贵了"：反问现有人力和管理成本；转向价值讨论（长期一致性、管理审计）。
  • "员工不会用 / 抗拒"：强调决策者是管理层；日常操作仅需倒 hopper、偶尔擦传感器；外包 / 流动工种反而更适合用机器（因为人员不 consistent，重新培训成本高）。
  • "有些角落搞不定"：承认边界——机器人负责大面积、规则区域，人工专注复杂角落与增值服务，形成分工。
  • "老美只爱一键搞定"：用逻辑 + 现场效果击穿——先把他们带到包房 / 宴会厅等 MT1 优势场景，再谈 ROI。不要让客户把你带到 MT1 劣势场景。

第三部分：现场部署与操作培训（含关键操作点）

3.1 标准部署流程（七步法）

  • ① 建图（Mapping）：将机器人放在预定 Home Point，APP 开始建图，手动推动机器人遍历所有需清洁区域边界。推车时注意：只能往一个方向推，不可反复来回（轮毂电机出力方向有限制）。
  • ② 地图编辑与风险设置：擦除噪点；设置虚拟墙 / 禁区（玻璃幕墙、花瓶、楼梯口）；对雷达看不到的物体（如玻璃门）手动画墙。
  • ③ 划定清洁区域：框选独立区域，为每个区域选择清扫模式（弓形 Bow / 回形 S-Shape）。
  • ④ 创建清洁任务：设置边刷 / 滚刷速度、吸力（默认 Medium）、移动速度、AI Adaptive 等参数。
  • ⑤ 设置排程（Scheduling）：将任务添加到时间表（每天 / 每周自动执行）、配置勿扰时段、设置重复次数。
  • ⑥ 试运行与调试：首次运行观察路径 / 避障，若某处徘徊或误识别，通过 Algorithm → Sensor Off 设置"传感器屏蔽区"修正。
  • ⑦ 验收与数据上传：连接 WiFi / 热点上传 user data，用于后续 ROI 证明与复购推动。

3.2 关键操作要点：确保吸尘头 / 滚刷正确就位

⚠ 这是演示成功的物理前提——团队必须反复练习，避免现场"空跑"。
  • 放下（工作状态）：手动下压或释放锁扣，使底部清洁组件"咔哒"一声落地；吸尘头可调高度以适应不同地毯（长毛放高档、短毛放最低）。
  • 抬起（运输状态）：向上抬起并锁定底盘，万向轮着地，像拉行李箱一样拖动。
  • 在位检测：底盘 + 吸尘头有检测传感器，未放下时 APP 启动任务会报错。
  • 演示要点：每次任务开始前口头确认"吸尘头已放下"；转场切换任务时要再次检查。

3.3 路径规划与 AI 特性

  • Bow（弓形 / 覆盖清洁）：沿清洁宽度（≈ 70 cm）规划平行路径，适合大面积全覆盖。
  • S-Shape（回形）：不断蚕食边缘的覆盖算法，路径可解释性较差——客户可能觉得"傻不拉几"；适合追求极致 coverage 的场景。
  • Spot Cleaning（巡检清洁）：宽度按 2.5 倍清洁宽度（≈ 1.5 m）规划，效率 6,000 m²/h，哪脏扫哪，不追求全覆盖。
  • AI Adaptive：在路径允许的偏移范围（约 40 × 40 cm）内，识别到垃圾会微调方向去吸，吸完回归原路径。
  • AI Cleaning Posture：识别到垃圾后改变朝向清洁。注意：部分客户喜欢（"好聪明"），部分客户会认为"哎你怎么转弯不扫那里了"——现场根据客户反应决定开关。
  • Teaching Path（贴边 / 转场路径）：两种用途——① 贴边清洁；② 跨区转场（机器跑完 A 区自动抬刷按 teaching path 移动到 B 区开始下一个任务）。

3.4 关键点 / 区域概念

  • Home Point：回家点 / 充电桩定位点，切换地图时机器必须对着 Home Point 重新定位。
  • Maintenance Point：故障点——当边刷 / 滚刷缠绕报错时，机器人回到此点等待人工处理，避免原地傻站。
  • 速度限制区：在视野盲区（转角、门口、柱子后）设置，避免高速出撞人。
  • Sensor Off / 传感器屏蔽区：当机器人在某区域反复打转，经三色界面确认是误识别（如花纹地毯被 AI 判成障碍物）时，在该区域屏蔽特定传感器（雷达 / RGB-D / 防撞条 / AI Visual）。调试前先看：橙色点 = 雷达 + RGB-D 实时障碍；蓝线 = 墙；带编号的框 = AI 识别物。
  • Hazardous 区域：如泳池边、扶梯口——系统自动将判定距离从 30 m 收紧到 6 m，避免反光 / 黑洞误判。泳池场景需单独部署 CG one。

3.5 新增高阶功能

  • 呼叫器（Call Device）：清洁工在某区域发现脏污时一键召唤，机器人自动前往该区域执行清洁任务。需预先绑定"呼叫器 — 清洁区域"。
  • 双机协同：两台机器人处于同一地图且进场距离在 ~10 m 内时，可通过通讯模块互相协作（如 MT1 扫完后召唤 CG one 洗地）。
  • 多地图拼接：停车场等大型场景支持多地图拼接；但 demo 时不建议，部署费时且容易变"打黑工"。
  • Mission Start / Resume 逻辑：点起始点模式需机器人在起点；选 Roof 模式机器人可在区域内任意位置开始任务。

3.6 日常操作与维护

  • 每日（~2 分钟）：倒 hopper、用干擦镜布擦一下传感器 / 摄像头。
  • 每 1 – 2 周：大清理 hopper、检查边刷 / 滚刷缠绕物、清空 dust bag。
  • 每 3 个月：清洁或更换 HEPA 滤网（可水洗复用）、检查胶条磨损。
  • 年度：Service Trip 上门检查主板、雷达、RGB-D、主要机械件；按使用强度评估是否更换电池（2,000 循环后）。

3.7 常见故障与处理

  • ① 定位丢失 / 选错地图：让客户对着 Home Point 重新 localize；每张地图对应独立的 Home Point。
  • ② 被关门 / 电量耗尽出不来：现场人工救援 + 手动线充或推回充电桩。
  • ③ 缠绕 / 滚刷烧损：一般因打包带、长毛发、不合规垃圾；通知现场人员清理 → 远程指导更换耗材。
  • ④ 效果变差：最常见——胶条破损留漏痕、刷子磨损、HEPA 堵塞 → 到期更换耗材。
  • ⑤ 硬件损坏：屏幕 / 雷达 / 摄像头被撞碎——现场换件。
管理提示：通过 APP Portal 监控 Error Report，主动打电话提醒客户做 Preventive Maintenance，客户会觉得"卖给我不是就不管了"，有助于年度 Service Package 续费。

第四部分：售后 / 运营层面的长期关系建设

  • "效果即服务"：销售不是卖硬件，而是卖长期清洁效果。MT1 + 耗材订阅 + 定期 Service Trip + 远程监控 = 四件套。
  • 用户数据反哺销售：拉取 user data 展示"每天任务数 / 时长 / coverage"，让客户看到"用得越多越值"，推动复购与加购（如增配 CG one 洗地）。
  • 清洁报告可视化：coverage 图 + trash map + 任务时长，通过手机 APP 实时推送给酒店 supervisor，替代签到表。
  • 外包清洁公司渠道：物业公司 / 清洁外包商是高优先级 B 端客户——他们自己就在为"人员流动性高、地形不熟悉、一致性差"头疼。MT1 + 固定排程 = 一致性保障。
  • 美国市场特殊性：疫情后劳动力短缺 + 人力成本高 + 签到表 / 督导文化成本高，整体有利于机器人替代。参考客户：Compass 等物业公司。

第五部分：总结与后续行动

  • 团队实操任务：在客户演示前，每人完整走一遍部署流程——建图、虚拟墙、清洁区、任务创建、Schedule、调试，反复练习"放下吸尘头"这一动作。
  • 客户材料准备：MT1 产品彩页、ROI 计算模板（含人力成本 vs 设备 TCO）、已部署客户视频（酒店宴会厅清洁 case）。
  • 演示道具：自备面粉 / 爽身粉 / 咖啡渣（撒一溜）、模拟碎屑（纸片），在现场不够脏时主动制造演示场景。
  • 现场策略：先把客户带到 MT1 优势场景（宴会厅、大通道），再谈 ROI；不要让客户把你带到劣势场景（小角落、楼梯）。
  • 演示目标：激发兴趣 → 建立信任 → 引导出深度评估 / 试点需求，而非当场成交。
  • 长期服务思维：硬件 + 耗材订阅 + Service Package + 远程支持，建立年度续费客户关系。
